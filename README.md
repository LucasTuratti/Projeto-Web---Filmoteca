# 🎬 Minha Watchlist Inteligente

Aplicação web completa para gerenciar sua lista de filmes e séries, 
desenvolvida com PHP, MySQL, HTML, CSS e JavaScript como projeto acadêmico 
de Desenvolvimento Web Full-stack.

---

## 📋 Sobre o Projeto

Sistema que permite ao usuário criar uma conta, fazer login e gerenciar 
sua watchlist pessoal de conteúdos audiovisuais, com filtros, avaliações 
e marcação de favoritos.

---

## ⚙️ Funcionalidades

- Cadastro e autenticação de usuários com senha criptografada
- CRUD completo de itens (adicionar, visualizar, editar e excluir)
- Filtros por status, gênero, favorito e título
- Ordenação por título e avaliação
- Avaliação pessoal com nota de 0 a 10
- Marcação de favoritos
- Link para trailer
- Estatísticas da watchlist (total, assistidos e favoritos)
- Proteção de páginas com verificação de login por sessão
- Edição de perfil do usuário

## 📚 Conceitos Aplicados

- Autenticação com sessões PHP e criptografia de senha (`password_hash`)
- Banco de dados relacional com MySQL e queries preparadas (prevenção de SQL Injection)
- CRUD com PHP e MySQLi
- Estilização com Tailwind CSS e CSS customizado
- Responsividade para dispositivos móveis
- Feedback visual com mensagens de sucesso e erro

---

## 🗄️ Banco de Dados

Duas tabelas principais:
- `usuarios` — armazena nome, e-mail e senha criptografada
- `itens_audiovisuais` — armazena todos os dados dos itens vinculados ao usuário
